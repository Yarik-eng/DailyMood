## Завдання

### Обрана предметна область
Проєкт — простий застосунок для ведення щоденника настрою та звичок (DailyMood). Користувач додає щоденні записи про настрій, зберігає короткий опис та активності, переглядає статистику за останній місяць, а також відслідковує звички та місячні цілі.

### Реалізовані вимоги
- Рівень 1: Створено сторінки "Головна" та "Про нас" — виконано.
- Рівень 2: Додано мінімум дві додаткові сторінки (Journal, Statistics, Goals, Favorites) з меню та адаптивною версткою — виконано.

## Хід виконання роботи

### Підготовка середовища розробки
- Версія Python: 3.11 (встановлено через winget в ході роботи).
- Фреймворк: Flask (версія, встановлена з requirements: Flask>=2.0.0, фактично встановлено Flask 3.1.2).
- ORM: Flask-SQLAlchemy (Flask-SQLAlchemy>=3.0.0).
- Інші інструменти: virtualenv (створена `.venv`), pip, PowerShell для запуску та діагностики.

Короткі кроки по відтворенню середовища (для інструкції):
```powershell
Set-Location -Path 'C:\Users\PC\DailyMood'
python -m venv .venv
. .\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
python app.py
```

### Опис реалізованих сторінок

#### Головна сторінка
Головна сторінка (`index.html`) містить опис проєкту, навігаційне меню до інших сторінок та посилання на щоденник, цілі й статистику. Дизайн адаптивний, використовуються стилі з `static/css`.

#### Сторінка "Про нас"
Сторінка `about.html` містить інформацію про команду та короткий опис мети проєкту.

#### Журнал (Journal)
Сторінка `journal.html` дозволяє переглядати записи, додавати нові (через REST API `/api/journal`) та фільтрувати записи по даті / настрою.

#### Статистика
Сторінка `statistics.html` відображає дані за останній місяць: кількість записів, найпоширеніший настрій, середній категоріальний настрій та графік трендів (дані формуються на сервері та передаються у шаблон).

#### Цілі та звички
Сторінка `goals.html` та API `/api/habits` дозволяють створювати і відмічати щоденні звички і місячні цілі.

## Ключові фрагменти коду

### Маршрути в `app.py` (приклади)
```python
@app.route('/')
def index():
    return render_template('index.html')

@app.route('/api/journal', methods=['GET'])
def list_entries():
    entries = MoodEntry.query.order_by(MoodEntry.date.desc()).all()
    return jsonify([e.to_dict() for e in entries]), 200
```

### Ініціалізація БД (фрагмент)
```python
basedir = os.path.abspath(os.path.dirname(__file__))
db_path = os.path.join(basedir, 'data', 'dailymood.db')
app.config['SQLALCHEMY_DATABASE_URI'] = f'sqlite:///{db_path}'
db.init_app(app)

with app.app_context():
    db.create_all()
```

## Висновки
- Успішно реалізовано: мультисторінковий Flask-застосунок з REST API для журналу, статистикою та базовими сторінками. Створено адаптивні шаблони та базу даних SQLite.
